# web-vitals-demo

This is the demo project for improve web vital metrics. It's a simple site with a lot of bad practices.

## How to use

1. Clone this repo

2. Install dependencies

```bash
yarn install
```

3. Run the project

```bash
yarn dev
```

4. Open the project in your browser

```bash
http://localhost:3000
```
